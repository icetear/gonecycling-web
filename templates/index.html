🔑 Ohne Profil: Token / Sync (anonym)
  • Kein Konto, keine E-Mail, kein Passwort.
  • Der Server speichert nur verschlüsselte Daten – er kann deine Reisen nicht lesen.
  • Keine persönlichen Daten gespeichert.
  • Zugang nur über den geheimen Token – verloren = Daten weg.
  • Gerätewechsel: Token selbst übertragen.
👤 Mit Profil (Upgrade)
  • Anmeldung mit E-Mail und Passwort auf jedem Gerät.
  • Passwort vergessen? Zurücksetzen per E-Mail.
  • Name und E-Mail werden auf dem Server gespeichert.
  • Der Schlüssel zum Entschlüsseln liegt zur Wiederherstellung auf dem Server.
  • Keine anonyme zero-knowledge-Nutzung mehr für dein Konto.
